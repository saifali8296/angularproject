Super Admin

Key Features

  • Privilege to create and manage the Clinic Admin and Doctor Admin
  • Edit Clinic, Enable Clinic and Change Passwords
  • Create New Clinic
Clinic Admin

Key Features

  • Create and manage both Doctor and Patient Interface, and this interface belongs to Particular Clinic
  • Profile can be updated or edited at anytime
  • Can change the logo image of the clinic, Slogan can be seen by Doctors and Patients
  • Clinic Admin can view the list of Doctors and also add new Doctors and their details
  • Can set the working hours of Clinic.
    Status for availability will be shown in green tick mark, a red cross will be shown for non-availability
  • Clinic Admin can close and open patient file through iPad application
    Clinic Admin can Fix Appointements, Intake Forms, Exam Forms, Admin Forms, Billing Process, Weekly and Monthly Status Update
